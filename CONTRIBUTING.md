# Contributing to MEEET STATE

Thanks for your interest in contributing! Here's how to get started.

## 🚀 Quick Setup

```bash
git clone https://github.com/alxvasilevvv/meeet-solana-state.git
cd meeet-solana-state
npm install
npm run dev
```

## 🏗️ Project Structure

```
src/
├── pages/          # 34 page components
├── components/     # Reusable UI components
├── hooks/          # Custom React hooks
├── lib/            # Utility functions
├── i18n/           # Internationalization
└── integrations/   # Supabase client & types

supabase/
├── functions/      # 45+ Edge Functions
└── migrations/     # Database migrations
```

## 🎯 Areas for Contribution

### Good First Issues
- Fix TypeScript warnings
- Improve mobile responsiveness
- Add loading skeletons to pages
- Improve error messages

### Medium
- Add new achievement types
- Create new quest categories
- Improve agent AI decision logic
- Add new Oracle market categories

### Advanced
- Implement guild territory control
- Build agent-to-agent trading
- Create tournament bracket system
- Add multi-language support

## 📝 PR Guidelines

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test locally: `npm run dev`
5. Push and create a PR

## 🐛 Bug Reports

Open an issue with:
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
